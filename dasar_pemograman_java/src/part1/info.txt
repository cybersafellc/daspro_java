2. Java tools
    - JDK (javac, jvm, default library)
    - IDE (intelIJ, netbeans, vscode + ext)

3. How java works?
    - java memiliki 2 software utamanya untuk menjalankan program yaitu javac (java compiler) dan java / jvm (java virtual mechine).
    - pertama kita akan menulis source code java dengan mengikuti aturan sintax java
    - lalu source code di compile menjadi bytecode dengan menggunakan javac software
    - bytecode akan di eksekusi oleh jvm mulai dari tahap intepreter, penerjemahan dan runtime.
    - dari keseluruhan tahap di atas hasil akhirnya source code kita berhasil menginstruksi hardware computer CPU dan mendapatkan output.
